# 真空面起射抽样逻辑说明

这份文档专门解释 `strack` 当前“真空面起射”思路背后的抽样逻辑，以及它为什么适合真空泄漏主导问题。文档分成两层：

- 第一层：理论上为什么应该这样抽样
- 第二层：当前 `strack` 版本具体实现成了什么

对应实现主要见：

- [src/strack_geometry.f90](/d:/Strack/src/strack_geometry.f90)
- [src/strack_solver.f90](/d:/Strack/src/strack_solver.f90)
- [docs/user/ray_sampling_and_boundaries.md](/d:/Strack/docs/user/ray_sampling_and_boundaries.md)

## 1. 问题背景

对有真空边界的输运问题，边界条件是“零入流”：

```text
psi(r, Omega, g) = 0
```

其中：

- `r` 在真空边界上
- `Omega` 是指向几何内部的方向
- `g` 是能群

如果射线是从体内随机起射，那么起点和起始角通量都不是物理边界条件直接给出的，而是数值算法自己构造的。这样做当然可以收敛，但在强泄漏问题中，边界条件的信息会传得比较慢，或者需要更长的 `inactive distance`、更多的射线数和更细的平源区才能稳定下来。

真空面起射的核心思想是：

1. 起点直接放在真实真空边界上
2. 入射方向只在“朝向几何内部”的半空间里抽样
3. 起始角通量直接取零
4. 把整条从真空边界进入、最终再次泄漏出的物理特征线走完整

这样做的好处是，边界条件不是通过“长距离渐近消除”间接体现，而是直接写进每条特征线的起始条件里。

## 2. 边界测度是什么

要从真空边界上抽样特征线，首先要回答一个问题：

“一条穿过边界的小面元 `dA`、方向落在小立体角 `dOmega` 内的特征线，应该以多大的概率被抽到？”

答案不是简单的：

```text
dP ∝ dA dOmega
```

而是：

```text
dP ∝ mu dA dOmega
```

这里：

- `mu = Omega · n_in`
- `n_in` 是边界的内法向
- `0 < mu <= 1`

这就是经典的边界相空间测度，也就是所谓的 Lambert 余弦权重。

### 2.1 为什么会多出一个 `mu`

考虑一个边界面元 `dA`。在单位时间内，从方向 `Omega` 穿过这个面元进入几何内部的“射线通量”正比于该方向在法向上的投影。

几何上，斜着入射的特征线，单位方向束在边界上看到的有效投影面积更小；只有法向分量会真正贡献“穿越边界”的速率。所以通过面元的方向测度是：

```text
dM = mu dA dOmega
```

而不是简单的 `dA dOmega`。

如果把它类比成光学里的 Lambert 定律，会更直观：

- 越接近法向入射，`mu` 越大
- 越擦着边界掠入，`mu` 越小

因此，真空面起射不是“所有朝内方向等概率”，而是“越朝内法向的方向越容易被抽到”。

## 3. 为什么起点要在面上均匀抽样

边界测度是：

```text
dM = mu dA dOmega
```

对位置的边缘分布，可以把方向积分掉：

```text
dP(r) ∝ dA * integral_over_inward_half_space(mu dOmega)
```

而对一个给定平面边界面来说，

```text
integral_over_inward_half_space(mu dOmega)
```

是常数，不依赖于边界面上的具体位置。

所以位置边缘分布满足：

```text
dP(r) ∝ dA
```

这说明：

- 单个平面真空边界上，起点应该按面积均匀抽样
- `2D` 里则退化成按边长均匀抽样

如果有多个真空边界面，就应该先按面面积 `3D` 或边长 `2D` 选面，再在该面上均匀选点。  
这正是 `strack` 当前实现的逻辑。

## 4. 为什么方向要按余弦分布抽样

### 4.1 三维情况

固定边界上的一个点 `r`，方向条件分布来自：

```text
p(Omega | r) ∝ mu
```

归一化常数是：

```text
integral_over_inward_hemisphere(mu dOmega) = pi
```

所以三维条件方向分布是：

```text
p(Omega | r) = mu / pi
```

这就是余弦分布。

把方向写成相对于内法向的球坐标：

- `theta`：与内法向夹角，`0 <= theta <= pi/2`
- `phi`：方位角，`0 <= phi < 2pi`
- `mu = cos(theta)`
- `dOmega = sin(theta) dtheta dphi`

于是：

```text
p(theta, phi) = (cos(theta) sin(theta)) / pi
```

进一步可见：

- `phi` 是均匀的
- `mu = cos(theta)` 的分布满足 `p(mu) = 2 mu`, `0 <= mu <= 1`

因此可以直接采样：

```text
mu = sqrt(xi_1)
phi = 2 pi xi_2
```

然后构造方向：

```text
Omega = mu n_in + sqrt(1 - mu^2) * (cos(phi) t_1 + sin(phi) t_2)
```

其中 `t_1, t_2` 是边界切平面上的正交基。

这正是当前 `strack` 三维平面真空起射的实现形式。

### 4.2 二维情况

二维里，方向不再是球面上的立体角，而是一条单位圆上的角度参数。设：

- `theta` 是相对于内法向的夹角
- `-pi/2 <= theta <= pi/2`
- `mu = cos(theta)`

边界测度退化成：

```text
dM = mu dℓ dtheta
```

对固定边界点，方向条件分布是：

```text
p(theta | r) = cos(theta) / 2
```

因为：

```text
integral_{-pi/2}^{pi/2} cos(theta) dtheta = 2
```

如果令：

```text
eta = sin(theta)
```

则 `eta` 在 `[-1, 1]` 上均匀分布。因为：

```text
d eta / d theta = cos(theta)
```

所以：

```text
p(eta) = 1/2
```

当前 `strack` 的二维实现正是用这组变量：

```text
eta ~ Uniform(-1, 1)
mu = sqrt(1 - eta^2)
Omega = mu n_in + eta t
```

这里 `t` 是边界切向单位向量。

## 5. 为什么不能简单地“朝内方向均匀抽样”

如果误用“半空间内方向均匀分布”，那么三维里会变成：

```text
p(Omega | r) = const
```

这会高估掠入射方向，低估接近法向方向。结果是：

- 进入几何的 chord length 分布被改坏
- 边界附近各向异性被改坏
- 泄漏和回填平衡都会偏

换句话说，真空面起射本身不是“从边界扔射线进去”这么简单，而是必须抽对边界测度。

## 6. 为什么起始角通量必须取零

真空边界条件本身就是零入流：

```text
psi(0) = 0
```

对单群、定源、均匀截面的段内解析解，若沿特征线弧长记作 `s`，则：

```text
d psi / ds + Sigma_t psi = Q
```

解为：

```text
psi(s) = psi(0) exp(-Sigma_t s) + (Q / Sigma_t) * (1 - exp(-Sigma_t s))
```

如果是真空边界起射，那么：

```text
psi(0) = 0
```

于是：

```text
psi(s) = (Q / Sigma_t) * (1 - exp(-Sigma_t s))
```

这说明靠近真空边界时，角通量应当从零逐渐长起来，而不是一上来就取 `Q / Sigma_t`。  
对于薄 slab、强泄漏问题，这个差别会非常重要。

## 7. 为什么活跃段结束时还要继续追到自然泄漏

这是当前 `strack` 对真空面起射最关键的一个处理。

### 7.1 体内随机起射和真空面起射的本质差别

对体内随机起射来说：

- 起点本来就是算法人为选的
- `psi(0)` 也是算法人为猜的
- `distance_inactive` 和 `distance_active` 本质上是在构造一个“可统计的代表性活动段”

所以体内起射时，有限的 active length 是一种数值近似，可以理解。

但对真空面起射来说：

- 起点是真实物理边界
- `psi(0)=0` 是真实物理条件
- 射线从进入几何的那一刻起就已经是“物理特征线”

这时如果在活跃段结束时，射线还没有走到真正泄漏位置，却被算法强行停掉，那么相当于：

- 你不是在求原问题
- 你是在求“沿弧长走到某个固定距离就人为终止”的另一个问题

### 7.2 人为截断会漏掉什么

设一条真空起射特征线的真实退出长度是 `L_exit`，而算法只算到 `L_act < L_exit`。  
那么少算掉的是后半段的全部贡献：

```text
Missing contribution = integral_{L_act}^{L_exit} [segment contribution] ds
```

如果用更抽象一点的写法，对某个区域 `r`、某个能群 `g`，缺失项可以写成：

```text
M_miss(r, g) = integral_{L_act}^{L_exit} w_r(s) psi_g(s) ds
```

其中 `w_r(s)` 表示该弧长位置是否处在区域 `r` 内。

这个缺失项有几个特点：

- 永远不是负的“噪声”，而是系统性漏算
- 对长 chord 更严重
- 对开放、薄、泄漏主导问题更敏感

所以如果不把真空起射特征线追到自然泄漏，结果通常会系统性偏低，特别是：

- 通量被压低
- `keff` 被压低
- 下游区域或远离入射面的区域更容易受影响

这正是 `strack` 先前在 `slab_1d_1g` 上观察到的现象。

### 7.3 从统计测度角度再看一遍

真空面起射抽样，本质上采样的是“从真空边界进入几何的完整 chord 集合”。  
一条 chord 的统计样本，应该从：

- 真空入射点
- 一直到它真正出界

如果只取前半段，相当于改变了 chord length 分布。  
原本应当进入统计的长 chord 被系统性裁短，统计测度不再正确。

因此，对真空面起射来说，“走到自然泄漏”为的是保持完整 chord 样本的无偏性，而不只是“多走一段看看”。

## 8. 当前 `strack` 的实现逻辑

当前版本对平面真空边界的实现可以概括成：

1. 扫描 geometry 中的 surfaces
2. 只接受与 `ray_source` 盒子重合的 `x/y/z-plane` 真空面
3. `2D` 时按边长选面，`3D` 时按面积选面
4. 在被选中的面上均匀选点
5. 在朝内半空间按余弦分布选方向
6. 令 `psi = 0`
7. 先走 `distance_inactive`
8. 再走 `distance_active`
9. 如果射线仍在几何内，则继续追到自然泄漏

当前实现的限制也要明确：

- 只支持轴对齐平面真空边界起射
- 曲面真空边界目前还没有专门的 surface launch
- 若真空面与 `ray_source` 盒子不重合，也不会触发这套逻辑

## 9. 球形真空边界应该怎么做

球形真空边界在理论上没有障碍，只是当前版本还没实现。

### 9.1 球面上如何均匀抽点

设球心为 `c`，半径为 `R`。可按下面方式在球面上均匀抽点：

```text
z = 1 - 2 xi_1
phi = 2 pi xi_2
r = c + R * [sqrt(1-z^2) cos(phi), sqrt(1-z^2) sin(phi), z]
```

这样得到的是球面均匀分布，而不是在参数角上均匀导致的“极区堆积”。

### 9.2 球面处如何抽朝内方向

先算球面点的外法向：

```text
n_out = (r - c) / R
```

内法向是：

```text
n_in = -n_out
```

然后在该点局部切平面上构造正交基 `t_1, t_2`，再按和前面完全一样的三维余弦分布采样：

```text
mu = sqrt(xi_3)
varphi = 2 pi xi_4
Omega = mu n_in + sqrt(1-mu^2) * (cos(varphi) t_1 + sin(varphi) t_2)
```

因此，从理论上说，球形真空边界并不需要不同的边界测度；它只需要：

- 正确的曲面均匀点采样
- 正确的局部法向和切向基

### 9.3 为什么这部分值得后续实现

如果后面要做：

- 临界球
- 真空包壳球
- 外边界是曲面的深穿透/屏蔽问题

曲面真空面起射会比单纯体内起射更自然，也更容易直接体现边界条件。

## 10. 与 OpenMC 风格体内起射的关系

真空面起射并不是在说“体内起射一定错误”，而是在说：

- 对强泄漏问题，真空边界条件可以被更直接地编码进抽样过程
- 对薄体系、真空 slab、开放临界球这类问题，边界条件主导性很强，直接从真空边界起射常常更稳

OpenMC 的随机射线方法走的是另一条路线：

- 全域体内起射
- 起始角通量猜测
- `dead zone`/`active zone`
- 更成熟的体积估计和低差异采样

这条路线当然可以算真空边界问题，但它的“好不好算”会更依赖：

- inactive length 是否足够
- 样本覆盖是否足够均匀
- 几何和体积估计是否足够稳定

所以两条路线不是“对错”关系，更像是：

- 体内起射：通用、统一、便于和闭体系共用一套框架
- 真空面起射：更针对开放问题，边界条件更直接

## 11. 作为论文方向时可以强调什么

如果以后真要把这一块整理成论文，一个可能的切入点不是“真空边界也能起射射线”这么简单，而是：

1. 把边界相空间测度 `mu dA dOmega` 明确写成随机特征线的抽样分布
2. 说明它对真空泄漏主导问题的适用性
3. 分析“完整 chord 追踪到自然泄漏”与“固定 active length 截断”之间的系统偏差差异
4. 给出与体内起射策略的误差、效率和稳定性对比
5. 再推广到曲面真空边界和混合抽样策略

真正有意思的地方，可能不只是“换一种起点”，而是：

- 如何定义开放体系的无偏 chord 样本
- 如何把边界条件直接嵌入随机特征线测度
- 如何兼顾精度与效率

## 当前程序里的用户开关

现在 `strack` 已经把这套策略做成了显式输入选项：

```xml
<ray_launch_mode>auto</ray_launch_mode>
<boundary_epsilon_shift>1.0e-8</boundary_epsilon_shift>
```

- `auto`：有合适真空平面外边界时自动启用本文讨论的真空面起射
- `volume`：强制改回体内随机起射，适合做 A/B 对照；当前实现里，射线撞上真空/void 泄漏面后会镜像回域内并把角通量重置为零，以便把零入流条件重新带回后续路径
- `vacuum-surface`：强制启用真空面起射；如果当前模型不满足真空平面外边界条件，则直接报错
- `boundary_epsilon_shift`：控制起射后入内、穿面后和反射后的微小位移量，默认 `1.0e-8`

对 `vacuum-surface` 来说，`distance_inactive` 一般可以直接设为 `0`。原因很简单：它的起始角通量已经是精确的零入流边界值，不需要再靠 inactive length 去“洗掉”错误初值。

因此，本文中的“真空面起射”现在既可以作为自动策略使用，也可以作为一个可控的数值实验开关来单独测试。

## 12. 一句话总结

真空面起射之所以要：

- 在面上均匀抽样起点
- 在朝内半空间按余弦分布抽方向
- 并在活跃段后继续追到自然泄漏

并不是经验做法，而是分别对应：

- 边界位置边缘分布 `dP(r) ∝ dA`
- 边界方向条件分布 `dP(Omega | r) ∝ mu dOmega`
- 对完整真空入射 chord 集合做无偏采样

这三点缺一不可。
