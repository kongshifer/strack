# Validation

`validation/` 用来存放 `strack` 的回归验证算例、对应截面、算例说明和结果汇总。

当前目录规则是“每个算例一个文件夹”：

- `validation/<case_name>/<case_name>.xml`
- `validation/<case_name>/<case_name>_mgxs.xml` 或其他本地截面文件
- `validation/<case_name>/<case_name>.md`
- `validation/<case_name>/<case_name>_results.py`

如果你想先看中文总览，了解每个算例是干什么的、主要验证什么，可以直接看 [case_overview_zh.md](/d:/Strack/validation/case_overview_zh.md)。
