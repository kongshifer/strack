# unstructured_circle_square_2g

## 题目目的

验证 `strack` 在显式二维随机特征线模式下，对“正方形外包、圆形燃料内区”的两区两群本征值问题是否能给出合理的 `keff` 与区域通量。

## 模型说明

- 几何：`3.0 cm x 3.0 cm` 正方形，中心 `2.0 cm` 直径圆形燃料区，外部为慢化剂区
- 空间表示：使用 8 个精确 CSG 象限分区
- 维度：显式 `2D`，输入中使用 `<spatial_dimension>2</spatial_dimension>`
- 边界：外边界四周反射，`z` 向为二维等效反射包层
- 运行参数：`120` 循环，`18` 个非活跃循环，`2500` 条随机特征线历史

## 截面参数

燃料区：

- `Sigma_t = [1.96647e-1, 5.96159e-1]`
- `nuSigma_f = [6.203e-3, 1.101e-1]`
- `chi = [1.0, 0.0]`
- `Sigma_s = [[1.78e-1, 1.002e-2], [1.089e-3, 5.255e-1]]`

慢化剂区：

- `Sigma_t = [2.22064e-1, 8.87874e-1]`
- `nuSigma_f = [0.0, 0.0]`
- `chi = [1.0, 0.0]`
- `Sigma_s = [[1.995e-1, 2.188e-2], [1.558e-3, 8.783e-1]]`

## 参考结果

用户提供的文献表中，`MG-MCNP3B` 参考解为：

- 快群通量：燃料区 `1.0`
- 快群通量：慢化区 `1.80023`
- 热群通量：燃料区 `0.55211`
- 热群通量：慢化区 `1.08581`
- `keff = 1.174655`

## 当前 `strack` 结果

当前版本使用 8 区象限平源区后，得到：

- `keff = 1.177152`
- 与 `MG-MCNP3B` 参考值的绝对误差：`0.002497`

按四分区轨长加权合并后的区域平均通量，并以燃料区快群通量归一，可得到近似：

- 燃料区快群：`1.00000`
- 慢化区快群：`0.97535`
- 燃料区热群：`0.55614`
- 慢化区热群：`0.57735`

说明：

- 当前自动回归只把 `keff` 作为硬判据
- 区域通量已记录在结果文件中，后续可继续通过更细平源区或更强几何加速改进

## 验证判据
